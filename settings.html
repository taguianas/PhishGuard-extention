Google Safe Browsing
Checks every URL against Google's real-time database of phishing, malware, and unwanted software. Requires a free API key.

Get a free key at Google Cloud Console Enable the Safe Browsing API, then create an API key under Credentials. Free tier: 10,000 requests/day.

PhishTank
Community-curated phishing URL database. Checks suspicious URLs on demand. Works without a key (rate limited) or with a free API key for higher limits.

Register for a free key at phishtank.com.

VirusTotal
Checks suspicious URLs against 70+ antivirus and URL scanner engines. Catches zero-day phishing kits not yet in feed databases. Free tier: 4 requests/min, 500 requests/day. Requires a free API key.

Get a free key at virustotal.com. Sign in, go to your profile, and copy the API key.

Desktop Notifications
Show a system notification when a high-risk link is detected. Each domain is reported at most once per hour.
Domain Age Check
Queries the RDAP registry to determine when a domain was registered. Newly registered domains are a strong indicator of phishing campaigns.

URL Shortener Expansion
Follow links through bit.ly, t.co, tinyurl.com, cutt.ly, is.gd, ow.ly, rebrand.ly and 15 other known shorteners so heuristics see the real destination, not the short link. Requires permission to access any origin (for the redirect target), requested the first time you enable it. Results cached for 24 hours.
SIEM Webhook
POST every high-risk detection as JSON to a webhook URL so it can be ingested by Splunk HEC, Elastic, or a custom SOC endpoint. Each detection URL is sent at most once per hour (dedup window).

Must be HTTPS. The extension will request permission for this origin the first time you save a new URL.

Sent as the Authorization header on every POST. Leave blank if your endpoint authenticates via URL token or IP allowlist.

Example payload 
{
  "source": "phishguard",
  "version": "2.0.0",
  "timestamp": "2026-04-20T14:32:11.284Z",
  "detection": {
    "url":       "https://paypa1-secure.tk/login",
    "domain":    "paypa1-secure.tk",
    "score":     85,
    "riskLevel": "high-risk",
    "indicators": [
      { "score": 65, "label": "Brand 'paypal' in subdomain" },
      { "score": 20, "label": "TLD on malicious list" }
    ],
    "domainAge": null,
    "feedHits": {
      "openphish": false, "safeBrowsing": true,
      "phishTank": false, "urlHaus": false, "virusTotal": false
    },
    "allowlisted": false
  }
}
